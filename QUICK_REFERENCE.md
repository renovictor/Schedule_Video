## Workflow: Recording + Playback

```
1. Recording Tab
   в”ңв”Җ Set Start Time (use Fresh button)
   в”ңв”Җ Select Duration (30m default)
   в”ңв”Җ Choose Record Source (Screen 1 default)
   в”ңв”Җ Select Audio (Realtek default)
   в”ңв”Җ Click Start в–¶пёҸ (Green)
   в””в”Җ Click Stop вҸ№пёҸ (Red) when done

2. Display Tab
   в”ңв”Җ Click Browse... рҹ“Ӯ
   в”ңв”Җ Select recorded file
   в”ңв”Җ Click Play в–¶пёҸ (Green)
   в”ңв”Җ Adjust Volume рҹ”Ҡ (0-100%)
   в”ңв”Җ Adjust Brightness вҳҖпёҸ (0-200%)
   в””в”Җ Use Position Slider вҸұпёҸ to seek
```

## Tips & Tricks

### Quick Recording
```
1. Click Start в–¶пёҸ
2. Record video
3. Click Stop вҸ№пёҸ
вҶ’ File saved automatically
```

### Quick Playback
```
1. Recording Tab вҶ’ Note file path
2. Display Tab вҶ’ Click Browse рҹ“Ӯ
3. Select file вҶ’ Click Play в–¶пёҸ
4. Drag Position Slider вҸұпёҸ to navigate
```

### Adjust Audio
```
в‘  Default: 80% volume
в‘Ў Drag slider left: Quieter (lower %)
в‘ў Drag slider right: Louder (higher %)
в‘Ј 0% = Muted
```

### Adjust Video
```
в‘  Default: 100% brightness
в‘Ў Drag slider left: Darker (0-100%)
в‘ў Drag slider right: Brighter (100-200%)
в‘Ј 50% = Half brightness
```
