## 🔍 Troubleshooting Decision Tree

```
Problem: Video not showing during playback
├─ Check: VIDEO_PLAYBACK_GUIDE.md → Section "Video Not Displaying"
├─ Try: FFMPEG_SETUP.md → Install FFmpeg
└─ Verify: File format is MP4/AVI/MOV/MKV

Problem: No audio during playback
├─ Check: VIDEO_PLAYBACK_GUIDE.md → Section "No Audio During Playback"
├─ Try: Volume slider not at 0%
├─ Try: System volume not muted
└─ Verify: Audio device connected

Problem: Application won't run
├─ Check: INSTALLATION_CHECKLIST.md
├─ Verify: Python 3.8+ installed
├─ Try: pip install -r requirements.txt --force-reinstall
└─ Check: requirements.txt dependencies

Problem: GUI not colorful
├─ Check: gui.py is updated
├─ Restart: Application
└─ Verify: PySide6 installed (pip install PySide6)

Problem: Can't create recording
├─ Check: QUICK_START.md → Recording procedure
├─ Verify: Output file path is writable
├─ Try: Different file location
└─ Check: Disk space available
```
